CSS Clearfix

Khi 1 thẻ html cao hơn thẻ mẹ chứa nó và nó được có float left hoặc right
=> Thì nó sẽ bị trượt khỏi thẻ
=> Thẻ mẹ không còn chứa được thẻ con
vậy để giải quyết vấn đề này có 2 cách phổ biến
Cách 1: đặt thẻ mẹ thuộc tính overflow: hidden (Cách này này có nhược điểm)
Cách 2: đó là sử dụng class clearfix và đặt class clearfix này cho thẻ mẹ .clearfix {
overflow: auto;
}
.clearfix::after {
content: "";
clear: both;
display: table;
}
